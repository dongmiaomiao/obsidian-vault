## 一、决定薪资的六个因素

| 因素 | 关键点 |

|------|--------|

| 公司业务 | OA 系统利润薄 → 薪资低；AI 行业 → 薪资高。业务利润空间决定薪酬天花板 |

| 公司规模 | 规模大 ≠ 薪资高。大公司职责单一、定价固定；小公司一人多岗、反而可能更高 |

| 目标岗位职级 | 小公司看职级（主管/经理/高级/专家/总监），大厂看序列（P 专家线 / M 管理线） |

| 岗位薪资范围 | 来源于市场定价 + 公司标准 + 部门成本，不是标准化数字。**多给的 10K 绝对不是白给** |

| 面试表现 | 自我介绍+项目介绍完美 → 气势/自信/大脑兴奋度拉满 → 比平时聪明 30% |

| 岗位紧急程度 | 面试后详细给你讲业务 = 很急；问 HR 招了多久，一两周以内 = 四星紧急；**招越久越不急** |

## 二、薪资自我评定：按区间参考

| 薪资区间 | 涨幅参考 | 备注 |

|---------|---------|------|

| 8K-15K | ~30% | 基础区间 |

| 15K-22K | ~20% | 心理门槛卡在 19/21/22 |

| 22K-35K | **待定** | 受行业影响极大（新能源高薪是行业福利，出行业可能降） |

| 35K 以上 | 10%-20% | 按职级浮动 |

**拿到 20K+ 的三个条件：** 行业够好 + 面试表现好 + 运气（投的公司范围刚好）

## 三、薪资低于/高于对方范围时

### 你的薪资 ＜ 对方范围

- 先积累面试经验，总结了无数次之后才能拿下高薪岗

- 不是报得高就能拿到的
### 你的薪资 ＞ 对方范围

- 问自己：拿下了愿不愿意去？愿意去的理由是什么？

- **核心价值：低薪岗位是垫脚石。** 拿一个 offer 当底气，去谈下一个时气势完全不同

- 实操：实拿 17K offer，对外说 22K，跟下一家谈持平
---
## 四、HR 压价的两大策略

### 策略一：问出你之前薪资，用涨幅规则压价

- "公司规定涨幅 10%，只能给 17K"

- **致命错误：** 说"20 左右"或"20-22 之间"——给的永远是左、没有右

- ✅ 直接说具体数字：18 就是 18，19 就是 19
### 策略二：踩着你的底线给

- 你的真实底线 = 你要的薪资 − 20%

- 不爽不是因为差 1000 块，而是底线被践踏

- 应对：第一次报价就往上提（如要 22K），留出回旋空间

---
## 五、谈判实操要点

| 原则 | 说明 |

|------|------|

| 谈月薪不谈年包 | "我月薪 35K"而非"年包 35 万"。年包 = 给自己挖坑 |

| 底薪多，绩效少 | 绩效比例多数可谈，理想 85:15 或 80:20，不要 40:60 |

| 按 12 薪谈 | 多出来的 2-3 薪是公司福利，不是谈判筹码 |

| 注意五险一金差额 | 差额大时在月薪上要求补回（25K+ 岗位可争取） |

| 不超岗位范围 | 唯一例外：底薪很低且对方范围窄 |

## 六、三大禁忌

| 禁忌 | 后果 |

|------|------|

| 出尔反尔，临时涨价 | 直接毁 offer（相当于办事前一天加彩礼） |

| 迷信网上技巧，耍小聪明 | HR 面前是班门弄斧；谈判余地只在"对方非你不可"时存在 |

| 为 1000 块来回拉扯 | 很掉价；不如爽快表态"差 1000 没关系我真的很想来" |

另外：**不要硬扛 14 薪，不要挑战公司规则**（薪资发放跟税务稽查挂钩，不能一人破例）

---
## 七、面对突然的谈薪电话
```
步骤：

1. 迅速回顾面试表现 + 你能为对方创造的价值

2. 锚定底线（如 18K）和爽点（如 20K），要价 22K

3. 可以反问："我看咱们写的范围是 20-25……"

4. 报价："之前 18K，想要涨幅，22K 在咱们范围内。"

```

**HR 挂电话后的操作：** 冲去找部门领导确认"这个人要 22K，值不值？能不能接受？"

---
## 八、终极总结

> **谈薪不是跟对方谈，是跟自己谈。**

- 底线在哪里？爽点在哪里？

- 差 1000 块不改变命运，底线被踩才让人难受

- 无路可走时：先接 offer，利用入职窗口期继续面

- 低底薪者：把**所有收入（底薪+绩效+现金年终奖）÷ 12** = 真实月薪

- 市场客观评定你，但市场数字 = 表达 + 表现 + 行业 + 运气的综合结果

**薪资范围只是参考，不是你的上限。**
