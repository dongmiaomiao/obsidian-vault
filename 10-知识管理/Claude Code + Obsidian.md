# Claude Code + Obsidian 知识加工 SOP

> CC 负责批量拆解文字，Obsidian 负责分类存档调取。各干各的，别搞混。

---

## 一、工具分工

| Claude Code（拆解工） | Obsidian（货架） |
|---|---|
| 批量处理文字：切分、重排、加标签 | 分门别类、快速搜索调取 |
| 按指令格式化 | 标签自动生效、图谱可视化 |
| 不负责"替你记" | 不负责"替你加工" |

---

## 二、标准流水线

| 步骤 | 你做什么 | CC 做什么 |
|---|---|---|
| ① 投料 | 把原始笔记丢到 `00-收件箱/` | 读取文件 |
| ② 下指令 | 粘贴下面的**标准指令模板** | 按格式拆解加工 |
| ③ 入库 | 把 CC 输出的 `.md` 拖进对应文件夹 | 任务完成 |
| ④ 调用 | Cmd+O 搜关键词 | — |

---

## 三、CC 标准指令模板（直接复制）

```
【任务】帮我整理课程笔记，用于存入 Obsidian 知识库。

【处理要求】
1. 根据一级标题（一、二、三）或自然段落，拆分成独立的 .md 文件
2. 文件命名：课程主题_核心关键词.md（如 数据治理_数据盘点.md）
3. 每个文件顶部加 Frontmatter：
   ---
   tags: [课程主题, 关键词1, 关键词2]
   alias: [口语化别名1, 口语化别名2]
   ---
4. 内容用"三要素"重构：
   - 这是啥（一句话定义）
   - 有啥用（解决什么问题）
   - 怎么用（3-5 条干货，删废话）
5. 所有输出文件放入 `[已加工]_课程主题/`

【笔记内容】：
（把你的原始笔记粘贴在这里）
```

## 四、快捷指令（省时间）

| 场景 | 直接复制这句话发给我 |
|---|---|
| 面试突击 | "根据我关于 [XXX] 的笔记，生成 3 个面试官最爱问的问题 + 1 分钟口语化回答" |
| 写汇报 | "根据 [XXX] 笔记里的内容，帮我写一封向领导汇报进度的邮件，简短、有数据、有下一步计划" |
| 跟人解释 | "把 [XXX] 概念用生活场景（装修、做饭、搬家）讲一遍，我要讲给非技术同事听" |
| 出执行清单 | "把 [XXX] 知识点变成 2 周可执行清单，按天列，每步写清楚找谁、干什么、产出什么" |
| 自测掌握 | "基于 [XXX] 笔记生成 10 道填空题或判断题" |
