第2章：准备工作：检查是不是装好了花 1 分钟确认一切就绪

检查 CC Switch 是否在运行

打开 Mac 的「应用程序」文件夹，找到 CC Switch 双击打开。成功后在屏幕右上角菜单栏（顶部，时间旁边）会看到一个图标。
✅ 蓝色图标 = 正常运行  |  ❌ 没图标 = 没启动

检查 Claude Code 是否安装

打开「终端」（Terminal.app），输入以下命令按回车：

claude --version

如果看到输出 2.1.168 之类的版本号，就说明装好了。

检查网络能不能连上 DeepSeek

在终端里输入：

curl -I https://api.deepseek.com

如果看到输出里有 HTTP/2 200 或 HTTP/1.1 200，说明网络是通的。

第3章：打开 CC Switch 并切换供应商选好要用哪个 AI 服务

打开 CC Switch

点击 Mac 右上角菜单栏的 CC Switch 图标，弹出窗口。

切换到 DeepSeek

在窗口里找到 Claude 供应商 这一栏，点击选择 DeepSeek。选中后会显示为「当前使用中」。

重启终端（重要！）

关闭终端窗口，重新打开。这样才能让新的设置生效。

确认切换成功

在终端里输入以下命令验证：

echo $ANTHROPIC_BASE_URL

如果输出是 https://api.deepseek.com/anthropic，就说明切换成功了！

⚠️ 记住：每次在 CC Switch 里切换供应商后，都必须关闭终端重开，否则不会生效。
🚀

第4章：在终端里启动 Claude Code第一次启动 AI 编程助手

进入你的工作目录

在终端里输入（cd 是"进入目录"的意思）：

$ cd /Users/pengshiba/Documents/test
启动 Claude Code

输入以下命令按回车：

$ claude

看到 > 提示符就说明成功进入了对话模式！

第一次启动可能会问的问题

「是否信任此工作区？」 → 输入 y（是）
「是否允许使用工具？」 → 选 Allow（允许）

💡 小提示： 以后再次启动，直接打开终端输入 claude 就行，不用重复设置。

第5章：跟 Claude 对话干活在 > 后面直接说人话

看到 > 提示符后，直接打字跟它说就行：

> 你好，你是谁？
最简单的打招呼
> @xxx 这个文件讲了什么？
用 @ 让 Claude 读文件
> 帮我写一个 Python 脚本
直接说需求，它帮你写代码
> 把标题改成"首页"
让它帮你改代码
💡 记住两个符号：
@文件名 —— 让 Claude 读取某个文件
!命令 —— 在对话里直接执行终端命令（如 !ls 列出文件）

退出 Claude Code： 输入 /exit 或按 Ctrl + C

第7章：切换不同模型（省钱/省力）简单任务用 Flash，复杂任务用 Pro

你的 DeepSeek 供应商配了两个模型，可以随时切换：

CC Switch 显示的名称实际模型适合做什么速度 / 价格
Sonnet / Opus deepseek-v4-pro 重构代码、深度分析、写复杂功能 稍慢但更强
Haiku deepseek-v4-flash 解释代码、写注释、简单修改 快，便宜

怎么切换？ 在 > 提示符下输入：/model，然后选你想要的。

💡 省钱技巧
简单问题 → 用 Haiku（flash）
复杂问题 → 用 Sonnet（pro）
不确定 → 先用 Haiku，不行再换 Sonnet
🎯 选模型建议
"解释这段代码" → Haiku
"帮我写个新功能" → Sonnet
"重构整个项目" → Sonnet

第8章：常见问题 & 解决方法遇到问题先看这里

Q1：启动时报错 "No API key found" ▼

原因： CC Switch 没选对供应商，或者切换后没重启终端。

解决方法：

打开 CC Switch，确认选中 DeepSeek
关闭终端，重新打开
再试 claude
Q2：启动后卡住 / 没反应 / 一直转圈 ▼

解决方法：

  1. 按 Ctrl + C 中断
  2. 检查网络：curl -I https://api.deepseek.com
  3. 检查 CC Switch 是否正在运行
  4. 重启 CC Switch
  5. 再试一次
Q3：Claude 回答得不对 / 质量差 ▼

原因： 可能当前选的是 Haiku（flash）模型，能力较弱。

解决方法： 在 Claude Code 里输入 /model，切换到 Sonnet（pro）模型，再重试。

Q4：怎么知道 DeepSeek 花了多少钱？ ▼

在 CC Switch 窗口里通常可以看到用量统计。也可以去 DeepSeek 官网的控制台查看 API 消耗详情。

Q5：想换回 MiniMax 或其他供应商 ▼
打开 CC Switch，选择 MiniMax 或其他
关闭终端，重新打开
启动 claude
Q6：对话越来越慢 ▼

原因： 对话历史太长了。

解决方法： 输入 /clear 清空对话，重新开始。

Q7：卡住了，想强制退出 ▼

按 Ctrl + C（一次不行就多按几次）。如果还不行，直接关掉终端窗口重开。
